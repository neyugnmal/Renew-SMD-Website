# 🎓 Renew SMD Website

A redesigned and responsive website project for SMD (School of Media Design), focusing on improving user experience, modern UI, and structured content presentation.

---

## 📌 Overview

This project aims to **renew and enhance the SMD website interface**, making it more visually appealing, user-friendly, and responsive across devices.

The website showcases key information about SMD including:
- About SMD
- Academic programs
- Media & News
- Student Gallery
- Contact information

---

## 🎯 Objectives

- Improve UI/UX design based on modern web standards  
- Enhance navigation structure and accessibility  
- Optimize layout for multiple screen sizes (Desktop, Tablet, Mobile)  
- Create a clean and consistent design system aligned with SMD branding  

---

## 🚀 Features

- ✨ Responsive layout (Mobile-first approach)  
- 🎨 Modern UI with consistent color system (SMD purple theme)  
- 📂 Structured content sections (Hero, Info, Grid, Cards)  
- 🖱️ Interactive elements (hover effects, dropdown menu)  
- 🔍 Search bar UI  
- 📱 Mobile navigation menu (hamburger)  
- 📰 News & Events card system  
- 👨‍🏫 Lecturer showcase grid  

---

## 🛠️ Technologies Used

- **HTML5** – structure  
- **CSS3** – styling & responsive design  
- **JavaScript (Vanilla JS)** – interaction & toggle menu  
- **Bootstrap 5** – layout support  
- **Font Awesome** – icons  
- **Google Fonts** – typography
